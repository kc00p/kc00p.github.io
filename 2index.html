Turtle Project

About the Project

Using refractorized code, the scene includes calls to draw multiple hot air balloons, multiple clouds, and hills without the busyness of unrefractorized code. To draw the hot air balloons, the balloon itself is a polygon with a rectangle associated with it, being different sizes but in set positions throughout the sky. These use coordinates such as (-200,240) to place each polygon-rectangle pair. The clouds are made of multiple white circles that vary in size. To draw multiple clouds, draw_cloud functions are used describe location, color, and size. Draw-multiple_clouds uses t, location, size, and color, then further refractorized with simply (t, num_clouds=3). The rolling hills are drawn with a curve function, for x, y, length, curve_factor, segments, and color. The refractorized code called the draw_hills function once, using t and fill_color green. These small curves required dozens of functions, showing the necessity of refractorization.

Sample Code

The draw_cloud leads into the draw_multiple_clouds function using (t, x, y, size=1.0, fill_color "white") in each cloud. Each cloud is made of multiple circles, then using jump_to(t, x, y), clouds are placed throughout the image. Therefore, in the refractorized code, draw_multiple_clouds is used with t and num_clouds=3. 


        def draw_cloud(t, x, y, size=1.0, fill_color="white"):
            """draw a cloud made of multiple circles"""
            jump_to(t, x, y)
            t.pencolor("white")
            if fill_color:
                t.fillcolor(fill_color)
                t.begin_fill()
            
            positions = [(0,0), (size * 15, size * 10), (size * 30, 0), (size * 45, 0), (size * 60, 0)]
            
            for pos_x, pos_y in positions:
                jump_to(t, x + pos_x, y + pos_y)
                draw_circle(t, size * 20, "white")
            
            if fill_color:
                t.end_fill()
            
        def draw_multiple_clouds(t, num_clouds=3):
            """Draw multiple clouds at set positions"""
            jump_to(t, 200, 50)
            draw_cloud(t, 60, 40, 1.5, "white")
            jump_to(t,-210, -80)
            draw_cloud(t, -270, 0, 1, "white")
            jump_to(t, 100, 200)
            draw_cloud(t, 100, 200, 1, "white")
            jump_to(t, -200, 200)
            draw_cloud(t, -200, 200, 2, "white")
